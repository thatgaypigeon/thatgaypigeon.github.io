Imports

Basic syntax

Installed modules can be imported using the import syntax.

Pigeon
import module

Relative imports

Local files can be imported using the dot syntax.

main.pigeon
import .myModule
Files
- main.pigeon
- myModule.pigeon

Folder imports

Modules can be mass-imported using folder imports. This requires an __init__.pigeon file in the subfolder.

main.pigeon
import myFolder

m = myFolder.myModule
Files
- main.pigeon
- myFolder
  - __init__.pigeon
  - myModule.pigeon

Modules can also be imported from parent directories.

main.pigeon
import ..myModule
Files
- myFolder
  - main.pigeon
- myModule.pigeon

Other examples

main.pigeon
import ...myModule
Files
- myFolder
  - subFolder
    - main.pigeon
- myModule.pigeon
main.pigeon
import ..anotherFolder.myModule
Files
- myFolder
  - main.pigeon
- anotherFolder
  - __init__.pigeon
  - myModule.pigeon
main.pigeon
import ..anotherFolder.subFolder.myModule
Files
- myFolder
  - main.pigeon
- anotherFolder
  - __init__.pigeon
  - subFolder
    - __init__.py
    - myModule.pigeon

Aliases

Modules can be imported using aliases.

Pigeon
import myModule as m

Object imports

Any globally-scoped objects are available once imported.

main.pigeon
import .myModule

print(myModule.x)  # 1
Files
- main.pigeon
- myModule.pigeon
myModule.pigeon
x = 1

They can also be imported directly using from, excluding any unused objects elsewhere in the module.

myModule.pigeon
x = 1
y = 2
main.pigeon
from .myModule import x

print(x)  # 1
print(y)  # undefined
Files
- main.pigeon
- myModule.pigeon

Objects used by other imported objects aren't imported themselves.

myModule.pigeon
x = 1
y = 2

func add_x_y() -> int:
    return x + y
main.pigeon
from .myModule import add

print(add_x_y())  # 3
print(x)  # undefined
print(y)  # undefined

Multiple objects can be imported with commas (,).

myModule.pigeon
x = 1
y = 2

func add_x_y() -> int:
    return x + y
main.pigeon
from .myModule import add_x_y, x

print(add_x_y())  # 3
print(x)  # 1
print(y)  # undefined

All global objects can be imported with an asterisk (*).

myModule.pigeon
x = 1
y = 2

func add_x_y() -> int:
    return x + y
main.pigeon
from .myModule import *

print(add_x_y())  # 3
print(x)  # 1
print(y)  # 2
